# Utility library bundle for Playdate

## What's included

* Shorthand library
* Scene engine

## How to build / use

### On macOS/Linux

```console
$ cmake .
$ cmake --build . --target build_util_libs
```

### On Windows

Untested, but you should be able to use cygwin to run the above.
